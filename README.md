# Prompt Engineering Task 4 — AI Mini Game Dashboard

This project is a **frontend-only mini-game dashboard** built as part of a prompt engineering task. It demonstrates how AI-assisted development tools can generate clean, interactive UI systems from a single structured prompt.

The dashboard focuses on player interaction, score tracking, and smooth UI animations.

---

# Requirements

| Requirement | Description |
|---|---|
| Development Type | Single-prompt generation |
| Scope | Frontend only |
| Tools | Bolt.New / Lovable.dev / Orchid |
| Deployment | Must include live hosted link |

---

# Features

| Feature | Description |
|---|---|
| Player Display | Shows player name and current score |
| Score Increment | Button to increase score dynamically |
| Animated Progress Bar | Visual representation of score progression |
| Responsive Layout | Works on all screen sizes |
| Modern UI | Clean and minimal dashboard design |

---

# Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js |
| Styling | Tailwind CSS |
| Animations | Framer Motion |

---

# Project Workflow

```text
Prompt Input
      ↓
AI UI Generation
      ↓
Frontend Rendering
      ↓
Score Interaction
      ↓
UI Update + Animation
```
